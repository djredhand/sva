Custom tabbed items using jQuery.

Tabs are tracked by using hash, so you can link directly to any particular tab directly. Also, there is an option to set any tab as a default landing page. This option is given as an arg when calling the plugin.

Tab content is hidden visually using the HTML5 Boilerplate concept. This way tab content is still accessible to screen readers. Also, there is a CSS fallback if JS is not supported.
